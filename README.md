## 📖 Project Overview

This project recreates the epic atmosphere of Clash of Clans with immersive visuals, animations, interactive sections, and a premium gaming-inspired UI.

Players can explore:

- 🏰 Village Building
- ⚔️ Epic Battles
- 🛡️ Clan Wars
- 👑 Legendary Heroes
- 💎 Resources & Loot
- 📊 Live Statistics
- ⏳ Event Countdown
- 📱 Download Section
